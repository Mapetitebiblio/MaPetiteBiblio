# MaPetiteBiblio — PWA v4 (QR élèves)

Nouveau : QR par élève (sélection ultra-rapide) + page d’accueil avec bouton rose “📋 Copier l’adresse du site”.

## Déploiement (GitHub Pages)
1) Créer un dépôt public `mapetitebiblio`
2) Uploader tous ces fichiers à la racine
3) Settings → Pages → Deploy from branch → main / (root) → Save
4) Ouvrir : https://TON_PSEUDO.github.io/mapetitebiblio/home.html
